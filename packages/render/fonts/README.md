# Vendored OFL fonts for deterministic Typst rendering

Fonts are resolved via `--ignore-system-fonts` and `--font-path` in the Typst driver. Only the weights required for invoice templates are vendored.

## Inter 4.1 (OFL-1.1)

- Source: https://github.com/rsms/inter/releases/tag/v4.1
- Download: https://github.com/rsms/inter/releases/download/v4.1/Inter-4.1.zip

| File | Weight |
| --- | --- |
| `inter/Inter-Regular.ttf` | 400 |
| `inter/Inter-Medium.ttf` | 500 |
| `inter/Inter-SemiBold.ttf` | 600 |
| `inter/Inter-Bold.ttf` | 700 |
| `inter/LICENSE.txt` | OFL license |
